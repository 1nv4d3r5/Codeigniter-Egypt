Applications Idea

applications are located in www-root/system/application/views/apps/ every folder is representing an application you can run an application with the url www.your-site.com/index.php/admin/app/app-name app-name: your app folder name
applications could be anything , like the applications of the facebook .
it could be image processing app, text editor, user manager, sections manager, site manager, sound editor, etc...

application structure

any application has a json file contains the application config variables like this : {
"name":"Sections manager",
"ver":"1.0",
"author":"Blaze Boy",
"website":"http:\/\/blazeeboy.wordpress.com",

"perm":"$CI->vunsy->user->is_root()",
"pages":{
"View Sections":"view"
},
"index":"View Sections",

"css_files":[],
"js_files":[],

"show_toolbar":true,
"show_title":true
}
 i think it's self discriptive and you can make a config file like that, besides the config file we got the php files that has the application code ,
Note: pages : has the pages titles as keys and the pages fie name as value
