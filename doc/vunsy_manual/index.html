Vunsy CMS

Vunsy is a very easy CMS that built on a very simple idea like HTML all the XML languages
every website is consisting of pages every page may have subpages and so
every page has a permissions for to determine of it's displayable for that user or under any other conditions
every page has layouts that consists of cells like the containers of widgets and other contents of any website
example : tables, divs, templates of websites

layout cell may contain an array of layouts or widgets , any layout or widgets named as "Content" that has a multiple permissions and properties like view permission,Add widgets in it,edit it, delete it,if it is displayable in subsections. Widget is a type of content that is more like a function that take paramters from multiple types and use it to display content

vunsy support the idea of applications on the website every application has a JSON defination file and permissions to use

Features

  • add edit delete content in an easy way
  • software manager for applications and sofware installed on the framework
  • datamapper for database ORM using style
  • javascript frame works: Dojo+dijit+dojox and jquery with jquery UI
  • ready to use PHP framework for developing widgets and using DOJO with PHP
  • very easy plugable framework
