Widget Idea

widgets are the main website content make them simple bueatiful, they maybe gallery, login form, article, image, some HTML what ever you want
you can consider widget as a function that takes arguments from vunsy in the view mode then show the content depending on that args
for example if you want to display a rich text that entered from the designer you have to say that the widget needs that item and name it
the widget need to pass a JSON object like that if mode is config just like layouts but layouts pass the number of cells , the JSON objetc has a info key that is an object contains the variables needed key is the name of object and you have to specify the type and if you want a default value (optional) you can pass the object like that here we said we need information to run that widget and they are only one variable named text from the type editor ( richtext Editor ) and has the default value ( default text ) we passed the json object if the mode is config mode if the view mode ( else ) then we'll display the variable named text from the $info object
