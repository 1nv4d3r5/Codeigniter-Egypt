GUI Library

gui library is a library for some of common gui controls like buttons tabs accordion grid and so on
you can include it to your page witha code like that $ci =& get_instance();
$ci->load->library('gui');
then use any function like that $ci->gui->textbox('name','text value' ) function are very simple they return the html and add the requirement using the add_css and add_js functions to vunsy automatically so you only needs to get the html and echo it or chan it with another function
