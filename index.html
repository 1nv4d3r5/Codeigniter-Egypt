Egypt is the land of warriors :D, alright i was kidding Egypt is a new CMS to develop web applications, it contain a ready made Users, Users group, Pages tree database and a front-end to manage them and their contents and properties.
Database ready
ready made database for users, contents, groups and sitemap (sections)
Codeigniter Based
based on the latest stable version of codeigniter and patched with the latest security patches
Datamapper library
an easy development using latest version of datamaper ORM layer
Dojo UI
a large set of UI controls with dijit, and a clear creative claro theme
Ion Auth Library
A simple but powerful authentication system using ion auth, plus a datamapper models for users and groups and an editor application for groups and users
Infinite Sections
create an infinite level of sections in the site map, plus specify permissions for how can view this section
Theme Helper
a theme helper for more easy recording of the page requirement and page assembly, that makes extending a piece of cake
Application based
application based CMS like facebook approach, every application has about box and menu bar, plus a permission for who could use it.
YAML friendly
Egypt uses YAML configuration objects for the definition of contents requirements
